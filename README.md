## Description
QuickClick is a Fruit Ninja–style Unity 3D game where the player must click on falling food items while avoiding bombs and preventing more than X foods from falling. The game features three difficulty levels (Easy, Medium, and Hard) with increasing challenge. This project was developed as part of the Udemy course “Curso Completo de Unity: inicia en el mundo de los videojuegos”, focusing on core gameplay mechanics and player interaction in Unity.

## Features
- Fruit Ninja–style gameplay based on mouse clicking.
- Falling food items that must be clicked to score points.
- Bomb objects that instantly end the game when clicked.
- Life system allowing up to X missed foods before game over.
- Three difficulty levels (Easy, Medium, Hard) with increasing speed and challenge.
- Score tracking and basic game state management.
- Simple and clean UI focused on gameplay clarity.
- Project developed as part of the Udemy course *“Curso Completo de Unity: inicia en el mundo de los videojuegos”*.


## Technical requirements:
- **Unity Version:** 6000.3.0f1 or higher.
- **Render Pipeline:** URP.
- **Platforms:** PC.

| Action | Keys |
